## 1. 작업 요약
- identity(shell): POSIX `ps -o pgid=,lstart=`(pgid 먼저·lstart 나머지 — comm 공백 모호성 제거)+`comm=` 별도. Windows `Win32_Process` CreationDate(ISO)·ExecutablePath. lookup 실패(timeout)와 부재 구분(IdentityLookupError).
- reconcile 3중+검증: 서명 owner(readOwner) + identity 존재 + startTime(불투명)·exe·groupId 일치 → 통과 시에만 종료. 불일치→skip, lookup실패→indeterminate(owner 보존), leaderless→검증불가로 kill 안 함(A21b v0.6 갭).
- terminateTree: 모든 시그널 직전 verifyLeader(μs 윈도우)·SIGTERM→grace→재검증→SIGKILL·isTreeDead(groupAlive). EPERM=alive. Windows taskkill /T(재검증 후).
- cancel = 3중검증 통과 종료. 테스트 31개(실 프로세스 spawn/kill).
