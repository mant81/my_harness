## 작업 — 런처 스킬 (첫 실행 bootstrap)
- [ ] **소스 확보**: `harness-ui/` 없으면 플러그인 번들 템플릿을 **realpath 프로젝트루트 하위** 복사(심링크 거부). 있으면 재사용. **package root·`package-lock.json` lstat/realpath 심링크 거부** 후 진행
- [ ] **런타임 확인**: `node`=`process.execPath`·`npm`(Windows `npm.cmd`) **각각** 존재·버전, 절대경로 해소(PATH hijack 차단), 없으면 graceful 실패
- [ ] **의존성(RCE 게이트)**: `node_modules` 없으면 명시 동의 → 기본 **`npm ci --ignore-scripts`** + 번들 **package-lock 해시 대조(fail-closed)**. postinstall 필수 시 2차 "임의코드" 경고 동의. silent 금지
- [ ] **멱등**: raw pidfile 신뢰 금지 → **§4-A 서명 레지스트리 + 로컬 헬스체크**로 실행중 판정. 실행중이면 재spawn 안 함
- [ ] **토큰**: 브라우저 오픈 시 실행 supervisor에 **새 single-use bootstrap 요청**, **토큰 URL/쿼리/로그 미노출**
- [ ] **기동/오픈**: 정지 시 `npm run dev`(cwd=harness-ui realpath 고정·최소 env). URL=`new URL()` `http://127.0.0.1:<numeric>/`만. 오픈=macOS `open`·Linux `xdg-open`·**Windows `cmd.exe /d /s /c start "" <url>`**(execFile+엄격 argv)
